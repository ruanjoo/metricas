# Classification Metrics Calculator

## Project Description
This project calculates key classification model evaluation metrics from a confusion matrix, including:

- **Accuracy**
- **Sensitivity (Recall)**
- **Specificity**
- **Precision**
- **F1-Score**

## Usage

Clone the repository:

```bash
git clone https://github.com/your-username/classification-metrics.git
cd classification-metrics
```

## Requirements
- **Python 3.10**
- **NumPy**
- **pandas**
